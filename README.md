# NEPSE Trade Checker — Daku's Tool

My personal trading decision tool for NEPSE.

I built this simple checklist because I was getting confused between too many indicators and emotions. This tool helps me follow my own rules consistently.

### Features
- Buy checklist (Fundamentals + Technical + Market)
- Sell checklist
- P/L calculator
- Automatic verdict (Strong Buy / Cautious / Do Not Buy)
- Critical blockers (EPS negative, new IPO, etc.)

### How to Use
1. Open `nepse-checklist-v1.html` in any browser (no installation needed)
2. Enter stock symbol and current price
3. Tick the boxes honestly
4. Follow the final verdict

**Note**: This is just a personal tool for my own trading. Not financial advice.
